For positive integers $n > m$, the triples $\paren{n^2 - m^2, 2 n m, n^2 + m^2}$ are Pythagorean triples,
with different pairs $\paren{n, m}$ yielding different triples.
The requirement $n^2 - m^2 = 2 n m \pm 1$ can be rewritten as $\paren{n + m}^2 = 2 n^2 \pm 1$.
Notice that this automatically implies $m < n$ (since otherwise $\paren{n + m}^2 \geq 4 n^2 > 2 n^2 + 1$).

Let $S_{+}$ be the set of pairs $\paren{r, s} \in \mathbb{N}^2$ such that $r^2 = 2 s^2 + 1$,
and $S_{-}$ be the set of pairs $\paren{r, s} \in \mathbb{N}^2$ such that $r^2 = 2 s^2 - 1$.
Observe that if $\paren{r, s} \in S_{+}$, then $\paren{r + 2 s, r + s} \in S_{-}$,
and if $\paren{r, s} \in S_{-}$, then $\paren{r + 2 s, r + s} \in S_{+}$:
\begin{itemize}
  \item[--] if $r^2 = 2 s^2 + 1$, then $\paren{r + 2 s}^2 = r^2 + 4 r s + 4 s^2 = 2 r^2 + 4 r s + 2 s^2 - 1 = 2 \paren{r + s}^2 - 1$;
  \item[--] if $r^2 = 2 s^2 - 1$, then $\paren{r + 2 s}^2 = r^2 + 4 r s + 4 s^2 = 2 r^2 + 4 r s + 2 s^2 + 1 = 2 \paren{r + s}^2 + 1$.
\end{itemize}
Since $\paren{1, 0} \in S_{+}$, there is an infinite sequence of distinct elements $\paren{r, s}$ in $S = S_{+} \cup S_{-}$.
For each of them, the pair $\paren{n, m} = \paren{r - s, s}$ determines a distinct Pythagorean triple of the desired form.
Each solution $\paren{a, b}$ to the given equation corresponds to the two Pythagorean triples $\paren{a, a + 1, b}$ and $\paren{a + 1, a, b}$,
so there are still infinitely many solutions $\paren{a, b}$.
