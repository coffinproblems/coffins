Let $c = 2 a + 1$.
The equation can equivalently be written as:
\begin{equation*}
  c^2 + 1 = 2 b^2
\end{equation*}
This is a Pell equation, which admits infinitely many integer solutions.
Note that $c$ is necessarily odd, since $c^2 = 2 b^2 - 1$ is odd;
so, in the corresponding solutions $\paren{a, b}$, $a$ is indeed an integer.
