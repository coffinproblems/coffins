For each $a \in \mathbb{Z}$,
let $\operatorname{P}\paren{a}$ be the set of prime divisors of $a$.
Characterise the set:
\begin{equation*}
  S = \setbuild*[\big]{\paren{a, b} \in \mathbb{N}^{2}}{
        \operatorname{P}\paren{a} = \operatorname{P}\paren{b}, \,
        \operatorname{P}\paren{a + 1} = \operatorname{P}\paren{b + 1}}
\end{equation*}

What about the set
$\displaystyle{
 T = \setbuild*[\big]{\paren{a, b} \in \mathbb{Z}^{2}}{
       \operatorname{P}\paren{a} = \operatorname{P}\paren{b}, \,
       \operatorname{P}\paren{a + 1} = \operatorname{P}\paren{b + 1}}
}$
?
