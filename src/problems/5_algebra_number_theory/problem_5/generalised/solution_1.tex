Yes to all.
\begin{enumerate}
 \item For any rational $a$ and integer $b$, $a^b$ is rational.
 \item An example is: $a = 5$, $b = \mfrac{1}{3}$, which give $a^b = \sqrt[3]{5}$.
 \item An example is $a = 2$ and $b = \log_2{3}$, which give $a^b = 3$.
 \item An example is $a = 2$ and $b = \mfrac{1}{2}\log_2{3}$, which give $a^b = \sqrt{3}$.
 \item An example is $a = \sqrt[3]{4}$ and $b = \mfrac{3}{2}$, which give $a^b = 2$.
 \item An example is $a = \sqrt{3}$ and $b = 3$, which give $a^b = 3 \sqrt{3}$.
 \item An exmaple is $a = \sqrt{2}$ and $b = 2 \log_2\paren{3}$, which give $a^b = 3$.
 \item An example is $a = \sqrt{2}$ and $b = \log_2\paren{3}$, which give $a^b = \sqrt{3}$.
\end{enumerate}
Note: $\log_2\paren{3}$ is irrational because no power of $2$ is a power of $3$ (with non-zero integer exponents).
