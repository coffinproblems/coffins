Let
$\alpha = \sqrt[3]{a \vphantom{b c} \mspace{2mu}}$,
$ \beta = \sqrt[3]{b \vphantom{c a} \mspace{2mu}}$,
$\gamma = \sqrt[3]{c \vphantom{a b} \mspace{2mu}}$.
Consider the polynomial identity:
\begin{equation*}
  x^3 + y^3 + z^3 - 3 x y z = \paren{x + y + z} \paren{x^2 + y^2 + z^2 - x y - y z - z x}
\end{equation*}
Let the polynomial $p\paren{x, y, z} = x^2 + y^2 + z^2 - x y - y z - z x$.
Then:
\begin{equation*}
  \frac{1}{\mspace{1mu} \alpha + \beta + \gamma \mspace{1mu}}
  = \frac{1}{\mspace{1mu} \alpha + \beta + \gamma \mspace{1mu}}
    \cdot \frac{p\paren{\alpha, \beta, \gamma}}
               {p\paren{\alpha, \beta, \gamma}}
  = \frac{p\paren{\alpha, \beta, \gamma}}
         {\mspace{2mu} \alpha^3 + \beta^3 + \gamma^3 - 3 \alpha \beta \gamma \mspace{2mu}}
  = \frac{p\paren{\alpha, \beta, \gamma}}
         {\mspace{2mu} a + b + c - 3 \sqrt[3]{a b c \mspace{2mu}} \mspace{2mu}}
\end{equation*}
The new denominator contains only one radical.
Let $u = a + b + c$ and $v = 3 \sqrt[3]{a b c \mspace{2mu}}$. Now:
\begin{equation*}
  \frac{1}{\mspace{2mu} \alpha + \beta + \gamma \mspace{2mu}}
  = \frac{\mspace{2mu} p\paren{\alpha, \beta, \gamma} \mspace{2mu}}
         {\mspace{2mu} u - v \mspace{2mu}}
    \cdot
    \frac{\mspace{2mu} u^2 + u v + v^2 \mspace{2mu}}
         {\mspace{2mu} u^2 + u v + v^2 \mspace{2mu}}
  = \frac{\mspace{2mu} \paren{\alpha^2 + \beta^2 + \gamma^2 - \alpha \beta - \beta \gamma - \gamma \alpha}
          \cdot \paren{u^2 + u v + v^2} \mspace{2mu}}
         {\mspace{2mu} u^3 - v^3 \mspace{2mu}}
\end{equation*}
Now the denominator is free of radicals,
and, in conclusion, the fraction can be written as:
\begin{equation*}
  \frac{\paren{ \!
          \sqrt[3]{a^2 \vphantom{b^2 c^2} \mspace{2mu}}
          + \sqrt[3]{b^2 \vphantom{a^2 c^2} \mspace{2mu}}
          + \sqrt[3]{c^2 \vphantom{a^2 b^2} \mspace{2mu}}
          - \sqrt[3]{a b \vphantom{c} \mspace{2mu}}
          - \sqrt[3]{b c \vphantom{a} \mspace{2mu}}
          - \sqrt[3]{c a \vphantom{b} \mspace{2mu}}
        } \! \cdot \!
        \paren{
          \paren{a + b + c}^2
          + 3 \paren{a + b + c} \sqrt[3]{a b c \mspace{2mu}}
          + 9 \sqrt[3]{a^2 b^2 c^2 \mspace{2mu}}
        }}
       {\mspace{1mu} \paren{a + b + c}^3 - 27 a b c \mspace{1mu}}
\end{equation*}
