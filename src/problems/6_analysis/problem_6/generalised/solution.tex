Using partial fraction decomposition:
\begin{equation*}
\begin{split}
  \sum\limits_{n=1}^{+ \infty} {\, \frac{1}{n^3 + 3 n^2 + 2n}} \,
  &= \, \sum\limits_{n=1}^{+ \infty} {\, \frac{1}{n \paren{n + 1} \paren{n + 2}}} \,
   = \, \frac{1}{2} \, \sum\limits_{n=1}^{+ \infty} {\, \frac{1}{n} - \frac{2}{n + 1} + \frac{1}{n + 2}} \, = \\[1.3ex]
  &= \, \frac{1}{2} \, \sum\limits_{n=1}^{+ \infty} {\, \frac{1}{n}}
     \, - \, \sum\limits_{n=2}^{+ \infty} {\, \frac{1}{n}}
     \, + \, \frac{1}{2} \, \sum\limits_{n=3}^{+ \infty} {\, \frac{1}{n}} \,
   = \frac{1}{2} + \frac{1}{4} - \frac{1}{2}
     \, + \, \sum\limits_{n=3}^{+ \infty} {\, \frac{1}{n} - \frac{1}{n}} \, = \\[1ex]
  &= \frac{1}{4}
\end{split}
\end{equation*}
